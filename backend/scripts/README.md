# Database Scripts

Утиліти для роботи з базою даних.

## check-db-connection.js

Перевіряє підключення до бази даних.

```bash
node scripts/check-db-connection.js
```

## create-database.js

Створює базу даних, якщо вона не існує.

```bash
node scripts/create-database.js
```

**Примітка:** Обидва скрипти автоматично завантажують `.env` файл з кореня проекту.
